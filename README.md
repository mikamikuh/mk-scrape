# mk-scrape

Clojureを使ってhtmlを読み込んで、Jekyll用の.mdファイルを出力するサンプル

mk-scrape.core ... 下記Namespaceのコードを実行する
mk-scrape.converter ... htmlファイルを読み込んで、Clojureのmapを作成する
mk-scrape.copy ... jpg/pngなど特定のファイルをコピーする
mk-scrape.export ... ファイルを出力する
